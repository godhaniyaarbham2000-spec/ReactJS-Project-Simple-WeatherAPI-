# 🌦 React Weather App Project

A clean and simple **React + Vite Weather Application** that fetches real-time weather data using OpenWeatherMap API.  
User can search any city and get instant weather details like temperature, humidity, and condition.

---

## 🚀 Features

* 🌍 Search weather by city name  
* 🌡️ Live temperature display  
* 💧 Humidity information  
* ☁️ Weather description with icon  
* ⚡ Loading & error handling  
* 🔐 API key stored in `.env` file  
* 📱 Responsive UI design  

---

## 🛠️ Tech Stack

* React.js (Hooks: useState)  
* Vite (Fast development build tool)  
* JavaScript (ES6+)  
* CSS3 (Custom styling)  
* OpenWeatherMap API  
* Environment Variables (.env)

## ⚙️ How to Run This Project

### 1. Clone the repository
```
git clone https://github.com/godhaniyaarbham2000-spec/ReactJS-Project-Simple-WeatherAPI.git
```

### 2. Move to project folder
```
cd ReactJS-Project-Simple-WeatherAPI
```

### 3. Install dependencies
```
npm install
```

### 4. Start development server
```
npm run dev
```
